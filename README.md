# My Store

## Descripción

Este proyecto es una aplicación de tienda en línea desarrollada con Next.js 14 y TypeScript. La aplicación permite a los usuarios crear cuentas, realizar compras y gestionar productos. Se ha utilizado Prisma como ORM y Neon.tech para la base de datos no relacional.
[Pagina de despliegue](https://my-store-ch6v.vercel.app/)

## Características

- **Creación de cuentas de usuario:** Los usuarios pueden registrarse y crear una cuenta en la aplicación según su rol.
- **Compra de productos:** Los clientes pueden ver y comprar productos disponibles.
- **Gestión de productos:** Tanto los clientes como los vendedores pueden leer los productos. Los vendedores pueden eliminar, editar y crear nuevos productos. Algunas funcionalidades de edición y creación presentan errores, pero las API están conectadas.

## Características

- **Edición de productos:** La funcionalidad de edición de productos tiene algunos errores.
- **CCreación de productos:** La creación de nuevos productos presenta algunos problemas, aunque las API están correctamente conectadas.
